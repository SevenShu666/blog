# 1.动态规划

如果某一问题有很多重叠子问题，使用动态规划是最有效的，动态规划中每一个状态一定是由上一个状态推导出来的

# 动态规划解题步骤

一些情况是递推公式决定了 dp 数组要如何初始化

  1. 确定 dp 数组以及下标的含义
  2. 确定递推公式
  3. dp 数组如何初始化
  4. 确定遍历顺序
  5. 举例推导 dp 数组

# 动态规划 debug

最好的方法将 dp 数组打印出来，将状态转移在 dp 数组上具体模拟一遍，确定最后推到出来的结果是不是想要的
