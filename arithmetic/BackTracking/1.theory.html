# 1.回溯算法理论基础

# 回溯法

回溯是递归的副产品，只要有递归就会有回溯。

递归的实现就是：每一次递归调用都会把函数的局部变量、参数值和返回地址等压入调用栈中

# 回溯效率

回溯的本质是穷举，穷举出所有可能，然后选出我们的答案，可以加一些减枝操作

# 回溯理解

回溯法解决的问题都可以抽象为树形结构

因为回溯法解决的都是在集合中递归查找子集，集合的大小就构成了树的宽度，递归的深度，都构成的树的深度。

# 回溯模板

返回值一般为void

参数需要写逻辑，然后需要什么参数就添加什么参数
