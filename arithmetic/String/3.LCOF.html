# 3.替换空格

力扣题目链接 (opens new window)

# 题目

请实现一个函数，把字符串 s 中的每个空格替换成"%20"。

示例 1：

输入：s = "We are happy."
输出："We%20are%20happy."

限制：

0 <= s 的长度 <= 10000

# 思路

先把字符串转换为数组，因为需要把空格替换成“%20”，那么字符串长度会增加n(空格长度)*2，先计算空格数量，再将数组长度加长n(空格长度)*2

  1. 增加双指针，left = s.length - 1,rigth = Array.length - 1

  2. 从后往前循环，遇到空格，将“%20”填入，不是空格就将字符填入

    • if (sArr[left] === " ") {
  sArr[right--] = "0";
  sArr[right--] = "2";
  sArr[right--] = "%";
} else {
  sArr[right--] = sArr[left];
}

# 代码

var replaceSpace = function (s) {
  const n = s.length;
  const sArr = Array.from(s);

  let count = 0;

  for (let i = 0; i < n; i++) {
    if (sArr[i] === " ") {
      count++;
    }
  }

  sArr.length = n + count * 2;

  let left = n - 1;
  let right = n + count * 2 - 1;

  while (left >= 0) {
    if (sArr[left] === " ") {
      sArr[right--] = "0";
      sArr[right--] = "2";
      sArr[right--] = "%";
    } else {
      sArr[right--] = sArr[left];
    }
    left--;
  }

  return sArr.join("");
};
